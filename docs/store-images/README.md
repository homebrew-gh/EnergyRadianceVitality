# Store Images

Put Zapstore / store-listing screenshots in this folder.

Recommended names:

- `screenshot-1.png`
- `screenshot-2.png`
- `screenshot-3.png`

Supported formats for the Zapstore config in this repo:

- `.png`
- `.jpg`
- `.jpeg`
- `.webp`

The repo's `zapstore.yaml` is configured to pick up screenshots from this folder automatically.
